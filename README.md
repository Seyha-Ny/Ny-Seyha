# Portfolio — Seyha Ny

Simple static portfolio inspired by the iPortfolio template. Uses light blue and teal colors as requested.

Open `index.html` in a browser to view the page.

Files added:
- `index.html` — main single-page site (contains Home / Resume / Contact sections)
- `css/style.css` — styles
- `js/main.js` — small animations for counters and progress bars

This site is a single-page layout: `index.html` contains the Resume and Contact sections inline (no separate pages required).

Notes:
- This is a static site; no build tool required.
- To change the copy, edit `index.html`.
